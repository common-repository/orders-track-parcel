== Description ==
Woocommerce Order Track Parcel enables merchants to add tracking links on completed
orders. The plugin can also notify the customer via email using Woocommerce Order Notes,
the email by default contains the tracking link provided for the shipment, once tracking 
link is added it will also show up on the order list view of the customer on the frontend.

== Installation ==

To Set it:

1.) Open the customer order look for the metabox "Track Parcels"
2.) Set the name of the Freight Company (Used as Button/Link Label) and set the fully qualified url of the tracking link on "Link" needs to include the http/https example: https://www.fedex.com or http://www.fedex.com is valid and www.fedex.com is invalid.

= Minimum Requirements =

* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress 4.8.2
* Woocommerce 3.2
